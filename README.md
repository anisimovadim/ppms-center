<h1>Инструкция к запуску</h1>
<p>Чтобы запустить проект, нужно установить необходимые модули:</p>

Node modules:
```shell
npm install
```
Vendor:
```shell
composer install
```

Полсе чего, можно запустить проект, командой:
```shell
php artisan serve
```
